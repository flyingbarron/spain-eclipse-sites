# User Features Guide

## Favorites & Personal Notes

This guide explains how to use the Favorites and Personal Notes features in the Spain Eclipse Sites Viewer.

---

## ⭐ Favorites

### Adding Sites to Favorites

1. **Browse sites** in the sidebar
2. **Click the ☆ icon** next to any site code
3. The icon changes to **⭐** to indicate it's favorited
4. The favorites count in the header updates automatically

### Viewing Your Favorites

1. Click **⭐ Favorites (N)** in the header
2. A modal opens showing all your favorite sites
3. Click any favorite to view its details
4. Click the **✕** button to remove a favorite

### Filtering by Favorites

1. In the sidebar, check **⭐ Favorites only**
2. The site list shows only your favorited sites
3. Uncheck to see all sites again

### Managing Favorites

**Export Favorites:**
- Click **📥 Export** in the Favorites modal
- Downloads a JSON file with your favorites
- Share with friends or backup your selections

**Import Favorites:**
- Click **📤 Import** in the Favorites modal
- Paste JSON data from a previous export
- Click **Import** to add them to your favorites

**Clear All:**
- Click **🗑️ Clear All** in the Favorites modal
- Confirms before removing all favorites

## 📝 Personal Notes

### Adding Notes to Sites

1. **View a site's details** by clicking on it
2. Scroll to the **Personal Notes** section
3. The section is **collapsed by default** if no note exists
4. Click the **▶** button to expand the notes widget
5. Type your notes in the text area
6. Click **💾 Save Note** to save

### Managing Notes

**Edit Notes:**
- Open the site details
- Expand the notes section if collapsed
- Edit the text and click **💾 Save Note**

**Delete Notes:**
- Click **🗑️ Delete Note** to remove the note
- No confirmation required - deletes immediately

**Filter Sites with Notes:**
- In the sidebar, check **📝 Sites with notes**
- Shows only sites where you've added notes
- Uncheck to see all sites again

**Note Indicators:**
- Sites with notes show a **📝** icon next to the site code
- Helps you quickly identify which sites have your notes

## 🔧 Advanced Filtering

### Accessing Advanced Filters

1. In the sidebar, click **🔧 Advanced Filters**
2. The advanced filter panel expands
3. Set your criteria and filters apply automatically

### Filter Options

**Tourist Value Range:**
- Set minimum and maximum tourist value (0-10)
- Only shows sites within this range
- Example: 5.0 to 10 for high-value sites only

**Cloud Coverage:**
- Drag slider to set maximum acceptable cloud coverage
- Example: Set to 30% to see only sites with <30% historical clouds
- Helps identify best weather prospects

**Bortle Scale (Darkness):**
- Drag slider to set maximum Bortle value (1-9)
- Lower values = darker skies
- Example: Set to 3 for excellent dark sky sites

**Route Difficulty:**
- Check boxes for acceptable difficulty levels:
  - Low
  - Medium
  - High
  - Very High
- Leave all unchecked to see all difficulties

### Resetting Filters

- Click **Reset Filters** at the bottom of advanced filters
- All filters return to default values
- Shows all sites again

## 💡 Tips & Tricks

### Efficient Site Selection

1. **Start with Eclipse Filter**: Check "🌑 Eclipse visible only"
2. **Add Cloud Filter**: Set max cloud coverage to 40%
3. **Check Darkness**: Set Bortle to 4 or less for dark skies
4. **Review Results**: Browse filtered sites
5. **Favorite the Best**: Click ☆ on promising sites

### Planning Your Eclipse Trip

1. **Favorite 10-15 sites** that meet your criteria
2. **Create a trip** from your favorites
3. **Organize by day** if visiting multiple sites
4. **Add notes** about timing, parking, facilities
5. **Export and share** with your travel group

### Combining Filters

**For the Perfect Eclipse Site:**
- ✅ Eclipse visible only
- ☁️ Cloud coverage < 30%
- 🌌 Bortle scale ≤ 3
- 🏔️ Horizon clearance: Clear only
- 🎯 Tourist value > 6.0

**For Easy Access:**
- 🚶 Route difficulty: Low or Medium
- 🏨 Near accommodations (use map view)
- 📍 Close to your starting point

### Backup Your Data

- **Export favorites regularly** to avoid losing selections
- **Export trips** before making major changes
- **Keep JSON files** in cloud storage for access anywhere

## 🔄 Data Persistence

All your favorites and trips are stored in your browser's **localStorage**:

- ✅ **Persists across sessions** - Data remains after closing browser
- ✅ **No account required** - Everything stored locally
- ✅ **Privacy-friendly** - No data sent to servers
- ⚠️ **Browser-specific** - Data doesn't sync across devices
- ⚠️ **Clearing browser data** will delete favorites and trips

### Syncing Across Devices

1. Export your favorites and trips as JSON
2. Save files to cloud storage (Dropbox, Google Drive, etc.)
3. On another device, import the JSON files
4. Your data is now available on both devices

## 🐛 Troubleshooting

**Favorites not saving:**
- Check if browser allows localStorage
- Try a different browser
- Export as backup before troubleshooting

**Filters not working:**
- Click "Reset Filters" and try again
- Refresh the page
- Check browser console for errors

**Import fails:**
- Verify JSON format is correct
- Ensure file is from this application
- Check for syntax errors in JSON

**Trip not showing sites:**
- Verify sites still exist in database
- Re-add sites if needed
- Check site codes are correct
