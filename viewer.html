ℹ️ About This Viewer

Spain Eclipse Site Viewer is a comprehensive tool for exploring potential viewing locations for the August 12, 2026 total solar eclipse in Spain.

Features

  • 🗺️ Interactive map with 100+ geological sites across Spain
  • 🌑 Eclipse visibility information for each location
  • ☁️ Historical cloud coverage data to help plan your viewing
  • 🏞️ Horizon profile views showing local topography
  • 🚗 Multi-site route planning with Google Maps integration
  • 📊 Detailed site information including tourist value and accessibility

Eclipse Details

The total solar eclipse on August 12, 2026 will cross Spain from west to east. The path of totality will pass through several regions, offering spectacular viewing opportunities for eclipse chasers worldwide.

❓ Help & Usage Guide

Getting Started

  • Browse Sites: Click on any site in the left sidebar to view details, images, and location information
  • Search: Use the search box to find sites by name or code
  • Filter: Toggle eclipse visibility and cloud coverage filters to narrow down your options

Understanding Site Information

  • 🌑 Eclipse Visible: Site is within the path of totality
  • ☀️ Low Cloud: Less than 30% average cloud coverage in August
  • ⛅ Medium Cloud: 30-60% average cloud coverage
  • ☁️ High Cloud: 60% or more average cloud coverage
  • Tourist Value: Rating of site's general tourist appeal (1-5)

Using the Map

  • Single Site: Click a site to view it on the map
  • Multiple Sites: Hold Ctrl (Cmd on Mac) and click multiple sites to plan a route
  • Route Planning: Drag sites in the route summary to reorder your itinerary
  • Export: Use the Google Maps or KML export buttons to save your route

Viewing Images

  • Click any image thumbnail to view full-size
  • Use arrow keys or navigation buttons to browse through images
  • Press Escape to close the image viewer

Tips for Eclipse Viewing

  • 🌤️ Choose sites with low cloud coverage for better viewing chances
  • 🏔️ Check horizon profiles to ensure clear views in the eclipse direction
  • 🚗 Plan your route in advance - roads will be busy on eclipse day
  • ⏰ Arrive early to secure a good viewing spot
  • 🕶️ Always use proper eclipse glasses or solar filters
